# image-resizer-and-compressor
A locally hosted web-based tool that allows the user to batch compress images, resize them to 550 Ã— 550px, strip all associated meta data, reduce pixel density to 96 dpi, and export them as individual JPG files for direct use on www.cwberry.com.

Node.js is required to run the application. The following command prompts need to be run in advance of operation:

npm init -y / npm install express multer sharp uuid / npm install -g nodemon / nodemon server.js / npm start
